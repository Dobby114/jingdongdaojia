# 京东到家仿写（移动端）
## 项目介绍
  京东到家仿写项目是一个基于vue3及其相关技术开发的即时零售品台，包括登录、注册、首页、首页搜索、商户详情、订单结算、购物车、订单管理、用户主页、地址管理等页面。
### 主要功能
* 注册、登录
* 单个商家详情展示
* 商品分类展示
* 购物车功能
* 订单提交
* 购物车、订单列表详情展示
* 首页商品搜索和搜索历史
